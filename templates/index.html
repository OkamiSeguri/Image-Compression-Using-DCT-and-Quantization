What happened during compression?

  • Color Space Conversion: Image converted to YCbCr color space
  • Block Division: Image split into 8x8 pixel blocks
  • DCT Transform: Each block converted to frequency domain
  • Quantization: High-frequency details reduced based on quality factor
  • Reconstruction: Inverse operations to rebuild the image

Tip: Lower quality = more compression but more visible artifacts (blocking, ringing). Higher quality = larger file but better visual fidelity.

Quantization Matrix

This 8x8 matrix determines how much each frequency component is divided. Larger values = more compression (more information lost). Top-left values are smaller to preserve low-frequency content (smooth gradients), while bottom-right values are larger to discard high-frequency details that humans are less sensitive to.

DCT Coefficients

Shows the frequency components of each 8x8 block. Brighter areas = higher energy. Top-left corner contains low frequencies (smooth areas), bottom-right contains high frequencies (edges/details).

Quantized Coefficients

Shows coefficients after quantization. Many values become zero (dark areas), especially high-frequency components. This is where compression happens - more zeros = better compression but more quality loss.
